SQL Server функция USER_NAME

В этом учебном пособии вы узнаете, как использовать функцию USER_NAME в SQL Server (Transact-SQL) с синтаксисом и примерами.

Описание

В SQL Server (Transact-SQL) функция USER_NAME возвращает имя пользователя в базе данных SQL Server.

Синтаксис

Синтаксис функции USER_NAME в SQL Server (Transact-SQL):

USER_NAME( [ identification_number ] )

Параметры или аргументы

identification_number - необязательный. Идентификационный номер, связанный с пользователем в SQL Server.

Примечание

  • Если указано identification_number, функция USER_NAME вернет имя пользователя для этого identification_number.
  • Если identification_number не указан, функция USER_NAME будет возвращать текущего пользователя на SQL Server в данном контексте.
  • См. Также функции CURRENT_USER, SESSION_USER, SYSTEM_USER.

Применение

Функция USER_NAME может использоваться в следующих версиях SQL Server (Transact-SQL):
SQL Server vNext, SQL Server 2016, SQL Server 2015, SQL Server 2014, SQL Server 2012, SQL Server 2008 R2, SQL Server 2008, SQL Server 2005

Пример

Рассмотрим некоторые примеры SQL Server функции USER_NAME, чтобы понять, как использовать функцию USER_NAME в SQL Server (Transact-SQL). Например:
