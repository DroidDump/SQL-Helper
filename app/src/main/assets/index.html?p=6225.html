PostgreSQL функция length

В этом учебном пособии вы узнаете, как использовать в PostgreSQL функцию length с синтаксисом и примерами.

Описание

Функция length PostgreSQL возвращает длину указанной строки, выраженную количеством символов.

Синтаксис

Синтаксис функции length в PostgreSQL:

length( string )

Параметры или аргументы

string
Строка, для которой возвращается длина.

Примечание

  • См. Также функции character_length и char_length.

Применение

Функция length может использоваться в следующих версиях PostgreSQL:

  • PostgreSQL 11, PostgreSQL 10, PostgreSQL 9.6, PostgreSQL 9.5, PostgreSQL 9.4, PostgreSQL 9.3, PostgreSQL 9.2, PostgreSQL 9.1, PostgreSQL 9.0, PostgreSQL 8.4
