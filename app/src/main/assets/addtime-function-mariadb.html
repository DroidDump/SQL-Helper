MariaDB функция ADDTIME

В этом учебном пособии вы узнаете, как использовать MariaDB ADDTIME функцию с синтаксисом и примерами.

Описание

MariaDB функция ADDTIME возвращает значение времени / даты и времени, после которого был добавлен определенный интервал времени.

Синтаксис

Синтаксис функции ADDTIME в MariaDB:

ADDTIME( start_value, time )

Параметры или аргументы

start_value
Значение времени или даты и времени, к которому следует добавить временной интервал.
time
Значение временного интервала, который вы хотите добавить. Это может быть положительное или отрицательное значение.

Примечание

  • Использование функции ADDTIME с отрицательным значением time в качестве параметра эквивалентно использованию функции SUBTIME.
  • См. также функцию SUBDATE.

Применение

Функция ADDTIME может использоваться в следующих версиях MariaDB:

  • MariaDB 10

--Результат: '01:15:28.999999'
 
SELECT ADDTIME('01:15:23.000001', '8:12:15.003441');
