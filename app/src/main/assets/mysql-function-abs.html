MySQL функция ABS

В этом учебном пособии вы узнаете, как использовать MySQL функцию ABS с синтаксисом и примерами.

Описание

MySQL функция ABS возвращает абсолютное значение числа.

Синтаксис

Синтаксис MySQL функции ABS:

ABS( number )

Параметры или аргументы

number - число, которое нужно преобразовать в абсолютное значение.

Применение

Функция ABS может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23
