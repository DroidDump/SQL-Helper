SQL Server функция ISDATE

В этом учебном пособии вы узнаете, как использовать функцию ISDATE в SQL Server (Transact-SQL) с синтаксисом и примерами.

Описание

В SQL Server (Transact-SQL) функция ISDATE возвращает 1, если выражение является допустимой датой. В противном случае он возвращает 0.

Синтаксис

Синтаксис функции ISDATE в SQL Server (Transact-SQL):

ISDATE( expression )

Параметры или аргументы

expression – выражение для проверки, является ли это датой.

Примечание

  • Функция ISDATE возвращает 1, если выражение является допустимой датой.
  • Функция ISDATE возвращает 0, если выражение НЕ является допустимой датой.

Применение

Функция ISDATE может использоваться в следующих версиях SQL Server (Transact-SQL):
SQL Server vNext, SQL Server 2016, SQL Server 2015, SQL Server 2014, SQL Server 2012, SQL Server 2008 R2, SQL Server 2008, SQL Server 2005
