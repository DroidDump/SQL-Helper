DROP LOGIN SQL Server

В этом учебном пособии вы узнаете, как использовать SQL Server DROP LOGIN с синтаксисом и примерами.

Описание

Оператор DROP LOGIN используется для удаления идентификатора (т.е.: Login), используемого для подключения к экземпляру SQL Server.

Синтаксис

Синтаксис оператора DROP LOGIN в SQL Server (Transact-SQL):

DROP LOGIN login_name;

Параметры или аргументы

user_name - имя Login для удаления.

Примечание

  • Вы не можете удалить Login, когда он в настоящий момент зарегистрирован в SQL Server.
  • Если вы удалите Login, на котором есть пользователи базы данных, то пользователи будут потеряны в SQL Server.
  • См. Также оператор CREATE LOGIN и оператор ALTER LOGIN.

Пример

Рассмотрим пример, показывающий, как использовать оператор DROP LOGIN в SQL Server (Transact_SQL). Например:

Transact-SQL
1
DROP LOGIN admin;

Этот пример DROP LOGIN приведет к удалению Login, с именем admin. Этот оператор DROP LOGIN будет запускаться только в том случае, если admin в настоящий момент не залогинился. Если admin в настоящее время залогинен в SQL Server, оператор DROP LOGIN вызовет ошибку.
