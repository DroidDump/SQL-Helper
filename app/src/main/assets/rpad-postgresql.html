PostgreSQL функция rpad

В этом учебном пособии вы узнаете, как использовать в PostgreSQL функцию rpad с синтаксисом и примерами.

Описание

Функция PostgreSQL rpad возвращает строку, дополненную c правой стороны указанной строкой определенной длины.

Синтаксис

Синтаксис функции rpad в PostgreSQL:

rpad( string, length, pad_string )

Параметры или аргументы

string
Строка для дополнения справа.
length
Длина результата после того, как string была дополнена справа.
pad_string
Указанная строка в правой части до string.

Примечание

  • Если длина строки превышает length, функция rpad удалит символы с левой стороны string, чтобы сократить ее до length символов.
  • См. Также функцию lpad.

Применение

Функцию rpad можно использовать в следующих версиях PostgreSQL:

  • PostgreSQL 11, PostgreSQL 10, PostgreSQL 9.6, PostgreSQL 9.5, PostgreSQL 9.4, PostgreSQL 9.3, PostgreSQL 9.2, PostgreSQL 9.1, PostgreSQL 9.0, PostgreSQL 8.4
