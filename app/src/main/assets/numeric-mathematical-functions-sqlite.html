Числовые/математические функции SQLite

Функция & Описание
ABS
SQLite функция abs возвращает абсолютное значение числа.
AVG
SQLite функция avg возвращает среднее значение выражения.
COUNT
SQLite функция count возвращает количество выражений.
MAX
SQLite функция max возвращает максимальное значение выражения.
MIN
SQLite функция min возвращает минимальное значение выражения.
RANDOM
SQLite функция random используется для возврата случайного числа.
ROUND
SQLite функция round возвращает число, округленное до определенного количества десятичных знаков.
SUM
SQLite функция sum возвращает суммированное значение выражения.
