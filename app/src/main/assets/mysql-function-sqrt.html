MySQL функция SQRT

В этом учебном пособии вы узнаете, как использовать MySQL функцию SQRT с синтаксисом и примерами.

Описание

MySQL функция SQRT возвращает квадратный корень числа.

Синтаксис

Синтаксис MySQL функции SQRT:

SQRT( number )

Параметры или аргументы

number - положительное число, используемое для вычисления квадратного корня.

Примечание

  • Функция SQRT будет возвращать NULL, если number является отрицательным значением.

Применение

Функция SQRT может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23

Пример
