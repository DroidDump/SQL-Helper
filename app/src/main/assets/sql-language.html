SQL - описание


Типы SQL-запросов

SELECTИзвлечение записей из таблицы
SELECT LIMITИзвлечение записей из таблицы и ограничение результатов
SELECT TOPИзвлечение записей из таблицы и ограничение результатов
INSERTВставка записей в таблицу
UPDATEОбновление записей в таблице
DELETEУдалить записи из таблицы.
TRUNCATE TABLEУдалить все записи из таблицы (без отката)
UNIONОбъединяет 2 набора результатов (удаляет дубликаты)
UNION ALLОбъединяет 2 набора результатов (включает дубликаты)
INTERSECTПересечение 2 наборов результатов
MINUSРезультирующий набор одного минус результирующий набор другого
EXCEPTНабор результатов одного минус набор результатов другого
JOINСоединение таблиц
DISTINCTПолучить уникальные записи
FROMСписок таблицы и JOIN информация
WHEREФильтр результатов
ORDER BYСортировать результаты запроса
GROUP BYГруппировать по одному или нескольким столбцам
HAVINGОграничить группы возвращаемых строк

Таблицы и представления SQL

CREATE TABLEСоздать таблицу
CREATE TABLE ASСоздать таблицу из определения и данных другой таблицы
ALTER TABLEДобавить, изменить или удалить столбцы в таблице; переименовать таблицу
DROP TABLEУдалить таблицу
GLOBAL TEMP TablesТаблицы, которые различны в рамках сеанса SQL
LOCAL TEMP TablesТаблицы, которые различаются в модулях и встроенной программе SQL
SQL VIEWВиртуальные таблицы (представления других таблиц)

SQL ключи, индексы

Primary KeysСоздание или удаление первичных ключей.
IndexesСоздание и удаление индексов (настройка производительности)

Операторы сравнения SQL

Comparison operatorsОператоры сравнения такие как =, !=, <>, >, >=, <, <= и т.д.

SQL условия

AND2 или более условия, которые должны быть выполнены
ORЛюбое из условий выполнено
AND И ORСочетание условий AND И OR
LIKEИспользуйте подстановочные знаки в предложении WHERE
INСоответствие выражения списку значений
NOTОтрицание
IS NULLПроверка значения на NULL
IS NOT NULLПроверка значения на NOT NULL
BETWEENВыборка в пределах диапазона (включительно)
EXISTSУсловие выполняется, если подзапрос возвращает хотя бы одну строку

Data TypesТипы данных в SQL

Псевдонимы SQL

ALIASESСоздание временного имени для столбца или таблицы

Литералы SQL

LITERALSЛитералы такие как string, integer, decimal и datetime.

Функции SQL

COUNTВозвращает количество выражений
SUMВозвращает сумму выражения
MINВозвращает минимум выражения
MAXВозвращает максимум выражения
AVGВозвращает среднее значение выражения
