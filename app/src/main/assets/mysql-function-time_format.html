MySQL функция TIME_FORMAT

В этом учебном пособии вы узнаете, как использовать MySQL функцию TIME_FORMAT с синтаксисом и примерами.

Описание

MySQL функция TIME_FORMAT форматирует время, указанное маской формата.

Синтаксис

Синтаксис MySQL функции TIME_FORMAT:

TIME_FORMAT( time, format_mask )

Параметры или аргументы

time - время форматирования.

format_mask - формат, применяемый ко времени. Ниже приведен список параметров для параметра format_mask. Эти параметры могут использоваться во многих комбинациях.

ЗначениеОписание
%fМикросекунды (от 000000 до 999999) %f доступны начиная с MySQL 4.1.1
%HЧас (от 00 до 23)
%hЧас (от 00 до 12)
%IЧас (от 00 до 12)
%iМинуты (от 00 до 59)
%pДо или после полудня AM или PM
%rВремя в 12-часовом формате AM или PM (hh:mm:ss AM/PM)
%SСекунды (от 00 до 59)
%sСекунды (от 00 до 59)
%TВремя в 24-часовом формате (hh:mm:ss)

Примечание

  • Функция TIME_FORMAT форматирует часы, минуты, секунды и микросекунды, найденные в значении времени.
  • См. также функцию DATE_FORMAT.

Применение

Функция TIME_FORMAT может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23

mysql> SELECT TIME_FORMAT('', '%H %i %s');
 
#Результат:  18 10 33                            
 
mysql> SELECT TIME_FORMAT('', '%h:%i:%s %p');
 
#Результат:  06:10:33 PM                            
 
mysql> SELECT TIME_FORMAT('', '%h:%i%p');
 
#Результат:  06:10PM                            
 
mysql> SELECT TIME_FORMAT('15:46:03.000001', '%r');
 
#Результат:  03:46:03 PM                          
 
mysql> SELECT TIME_FORMAT('15:46:03.000001', '%T');
 
#Результат:  15:46:03                            
 
mysql> SELECT TIME_FORMAT('05:46:03.000001', '%f');
 
#Результат:  000001
