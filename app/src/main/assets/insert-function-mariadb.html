MariaDB функция INSERT

В этом учебном пособии вы узнаете, как использовать MariaDB функцию INSERT с синтаксисом и примерами.

Описание

MariaDB функция INSERT вставляет подстроку в строку в указанной позиции на определенное количество символов.

Синтаксис

Синтаксис функции INSERT в MariaDB:

INSERT( string, position, number, substring )

Параметры или аргументы

string
Строка для изменения.
position
Позиция в string для вставки substring.
number
Количество символов для замены в string.
substring
Подстрока для вставки в string.

Примечание

  • Первая позиция в string - 1.
  • Если position находится за пределами длины string, функция INSERT вернет string.
  • Если number находится за пределами длины остальной строки, функция INSERT заменит string, начиная с position до конца string)

Применение

Функция INSERT может использоваться в следующих версиях MariaDB:

  • MariaDB 10
