MySQL функция WEEKDAY

В этом учебном пособии вы узнаете, как использовать MySQL функцию WEEKDAY с синтаксисом и примерами.

Описание

MySQL функция WEEKDAY возвращает индекс дня недели.

Синтаксис

Синтаксис MySQL функции WEEKDAY:

WEEKDAY( date_value )

Параметры или аргументы

date_value - значение даты или даты/времени, из которого извлекается индекс дня недели.

Примечание

  • Функция WEEKDAY возвращает индекс дня недели (0=Monday, 1=Tuesday, 2=Wednesday, 3=Thursday, 4=Friday, 5=Saturday, 6=Sunday), учитывая значение даты.
  • См. также функцию DAYNAME.

Применение

Функция WEEKDAY может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23
