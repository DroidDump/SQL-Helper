rtrim функция SQLite

В этом учебном пособии вы узнаете, как использовать SQLite функцию rtrim с синтаксисом и примерами.

Описание

SQLite функция rtrim удаляет все указанные символы из правой части строки.

Синтаксис

Синтаксис функции rtrim в SQLite:

rtrim( string, [ trim_string ] )

Параметры или аргументы

string
Строка для обрезки символов с правой стороны.

trim_string
Строка, которая будет удалена из правой части string. Если этот параметр пропущен, функция rtrim удалит все завершающие пробелы из string.

Примечание

  • Смотрите также функции ltrim и trim.

Применение

Функция rtrim может использоваться в следующих версиях SQLite:

  • SQLite 3.8.6, SQLite 3.8.x, SQLite 3.7.x, SQLite 3.6.x
