MariaDB функция PASSWORD

В этом учебном пособии вы узнаете, как использовать MariaDB функцию PASSWORD с синтаксисом и примерами.

Описание

MariaDB функция PASSWORD используется системой аутентификации в MariaDB для генерации хешированного пароля из строки открытого текста с использованием более мощных методов хеширования. Чтобы использовать более старые методы хеширования, используйте OLD_PASSWORD функцию.

Синтаксис

Синтаксис функции PASSWORD в MariaDB:

PASSWORD( string )

Параметры или аргументы

string
Текстовая строка пароля, которая является источником для создания зашифрованного/хешированного пароля в MariaDB.

Примечание

  • Функция PASSWORD вернет NULL, если string равна NULL.
  • Функция PASSWORD выполняет одностороннее шифрование.
  • Функция PASSWORD используется системой аутентификации в MariaDB для хранения паролей.
  • Не используйте функцию PASSWORD в вашем собственном приложении, вместо этого используйте функции MD5 или SHA1.
  • См. также функцию ENCRYPT.

Применение

Функция PASSWORD может использоваться в следующих версиях MariaDB:

  • MariaDB 10
