SQL ЛОКАЛЬНЫЕ ВРЕМЕННЫЕ ТАБЛИЦЫ

В этом учебном материале вы узнаете, как создавать SQL LOCAL TEMPORARY TABLES с синтаксисом и примерами.

Описание

SQL локальные временные таблицы - уникальные в модулях и встроенных программах SQL в сеансах SQL.

Синтаксис

Синтаксис для DECLARE LOCAL TEMPORARY TABLE в SQL.

DECLARE LOCAL TEMPORARY TABLE table_name
( column1 datatype [ NULL | NOT NULL ],
column2 datatype [ NULL | NOT NULL ],
...
);

Параметры или аргумент

table_name
Имя локальной временной таблицы, которую вы хотите создать
column1, column2
Столбцы, которые вы хотите создать в локальной временной таблице. Каждый столбец должен иметь тип данных. Столбец должен быть определен как NULL или NOT NULL, и если это значение оставить пустым, база данных принимает NULL в качестве значения по умолчанию

Пример

Давайте посмотрим SQL пример DECLARE LOCAL TEMPORARY TABLE .

DECLARE LOCAL TEMPORARY TABLE suppliers_temp
( supplier_id int NOT NULL,
  supplier_name char(50) NOT NULL,
  contact_name char(50)
);

Этот пример создаст локальную временную таблицу suppliers_temp.
