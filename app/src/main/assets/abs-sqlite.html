abs функция SQLite

В этом учебном пособии вы узнаете, как использовать SQLite функцию abs с синтаксисом и примерами.

Описание

SQLite функция abs возвращает абсолютное значение числа.

Синтаксис

Синтаксис функции abs в SQLite:

abs( number )

Параметры или аргументы

number
Число для преобразования в абсолютное значение.

Примечание

  • Функция abs возвращает NULL, если параметр number является значением NULL.
  • Функция abs возвращает 0.0, если параметр number является строкой или BLOB-объектом, который не имеет числового значения.

Применение

Функция abs может использоваться в следующих версиях SQLite:

  • SQLite 3.8.6, SQLite 3.8.x, SQLite 3.7.x, SQLite 3.6.x
