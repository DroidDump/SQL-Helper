ВКЛЮЧЕНИЕ ВСЕХ ТРИГГЕРОВ ТАБЛИЦЫ

В этом учебном материале вы узнаете, как включить все триггеры таблицы в Oracle c синтаксисом и примерами.

Описание

При наличии всех отключенных триггеров таблицы, вам может понадобиться снова их включить. Вы можете сделать это с помощью команды ALTER TABLE.

Синтаксис

ALTER TABLE имя_таблицы ENABLE ALL TRIGGERS;

Параметры и аргументы
имя_таблицы наименование таблицы, для которой включаются все триггеры.

Примечание

  • Смотрите также, как включить триггер.
  • Смотрите также, как отключить триггер таблицы или отключить все триггеры таблицы.

Пример

Рассмотрим пример, который показывает, как включить все триггеры таблицы в Oracle.

ALTER TABLE orders ENABLE ALL TRIGGERS;

Этот пример использует команду ALTER TABLE, для включения всех триггеров таблицы orders.
