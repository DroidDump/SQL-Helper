MariaDB функция DATE_FORMAT

В этом учебном пособии вы узнаете, как использовать MariaDB функцию DATE_FORMAT с синтаксисом и примерами.

Описание

MariaDB функция DATE_FORMAT форматирует дату в соответствии с маской формата.

Синтаксис

Синтаксис функции DATE_FORMAT в MariaDB:

DATE_FORMAT( date, format_mask )

Параметры или аргументы

date
Дата для форматирования.
format_mask
Формат, применяемый к date. Ниже приведен список параметров для параметра format_mask. Эти параметры могут использоваться во многих комбинациях.

%aСокращенное название дня недели (с Sun по Sat)
%bСокращенное название месяца(с Jan по Dec)

%DДень месяца в виде числового значения, за которым следует суффикс (1st, 2nd, 3rd, ...)
%dДень месяца в виде числового значения (от 01 до 31)

Примечание

  • Дневные и месячные диапазоны начинаются с 00, поскольку MariaDB позволяет сохранять даты неполными. Например: '2019-00-00'.
  • Смотрите также функцию TIME_FORMAT.

Применение

Функцию DATE_FORMAT можно использовать в следующих версиях MariaDB:

  • MariaDB 10
