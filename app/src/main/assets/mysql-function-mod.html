MySQL функция MOD

В этом учебном пособии вы узнаете, как использовать MySQL функцию MOD с синтаксисом и примерами.

Описание

MySQL функция MOD возвращает остаток от деления n на m.

Синтаксис

Синтаксис MySQL функции MOD:

MOD( n, m )

ИЛИ

n MOD m

ИЛИ

n % m

Параметры или аргументы

n - значение, которое будет делиться на m.

m - значение, которое будет делить n.

Примечание

  • Функция MOD использует формулу n / m, а остаток - то, что возвращается.
  • Начиная с MySQL 4.1.7, функция MOD возвращает точный остаток (без округления).
  • До MySQL 4.1.7 функция MOD округляет остаток до целочисленного значения.
  • Синтаксис MOD-функции n mod m был введен в MySQL 4.1.

Применение

Функция MOD может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23
