Числовые / математические функции SQL Server

Функция & Описание
ABS
В SQL Server (Transact-SQL) функция ABS возвращает абсолютное значение числа.
AVG
В SQL Server (Transact-SQL) функция AVG возвращает среднее значение выражения.
CEILING
В SQL Server (Transact-SQL) функция CEILING возвращает наименьшее целочисленное значение, которое больше или равно числу.
COUNT
В SQL Server (Transact-SQL) функция COUNT возвращает количество строк поля или выражения в результирующий набор.
FLOOR
В SQL Server (Transact-SQL) функция FLOOR возвращает наибольшее целочисленное значение, которое равно или меньше числа.
MAX
В SQL Server (Transact-SQL) функция MAX возвращает максимальное значение выражения.
MIN
В SQL Server (Transact-SQL) функция MIN возвращает минимальное значение выражения.
RAND
В SQL Server (Transact-SQL) функция RAND может использоваться для возврата случайного числа или случайного числа в пределах диапазона.
ROUND
В SQL Server (Transact-SQL) функция ROUND возвращает число, округленное до определенного количества десятичных знаков.
SIGN
В SQL Server (Transact-SQL) функция SIGN возвращает значение, обозначающее знак числа.
SUM
В SQL Server (Transact-SQL) функция SUM возвращает суммарное значение выражения.
