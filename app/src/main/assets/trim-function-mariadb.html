MariaDB функция TRIM

В этом учебном пособии вы узнаете, как использовать MariaDB функцию TRIM с синтаксисом и примерами.

Описание

MariaDB функция TRIM удаляет все указанные символы в начале или в конце строки.

Синтаксис

Синтаксис функции TRIM в MariaDB:

TRIM( [ LEADING | TRAILING | BOTH ] [ trim_character FROM ] string )

Параметры или аргументы

LEADING
Необязательный. Удаляет trim_character из передней части string
TRAILING
Необязательный. Удаляет trim_character из конца string.
BOTH
Необязательный. Удаляет trim_character из начала и конца string.
trim_character
Необязательный. Символ, который будет удален из string. Если этот параметр не указан, он удалит пробелы из string.
string
Строка для обрезки.

Примечание

  • Если вы не укажете значение для первого параметра (LEADING, TRAILING, BOTH), функция TRIM по умолчанию будет иметь значение BOTH и удалять trim_character как с начала, так и с конца string.
  • Если вы не укажете trim_character, функция TRIM по умолчанию удалит символы пробела.

Применение

Функцию TRIM можно использовать в следующих версиях MariaDB:

  • MariaDB 10

SELECT TRIM(LEADING ' ' FROM '  Google.com  ');
 
--Результат: 'Google.com  '
 
SELECT TRIM(TRAILING ' ' FROM '  Google.com  ');
 
--Результат: '  Google.com'
 
SELECT TRIM(BOTH ' ' FROM '  Google.com  ');
 
--Результат: 'Google.com'
 
SELECT TRIM(' ' FROM '  Google.com  ');
 
--Результат: 'Google.com'
 
SELECT TRIM('   Google.com   ');
 
--Результат: 'Google.com'
 
SELECT TRIM(LEADING 'a' FROM 'aaa987');
 
--Результат: '987'
 
SELECT TRIM(TRAILING '1' FROM 'Taxi11');
 
--Результат: 'Taxi'
 
SELECT TRIM(BOTH '123' FROM '123Taxi123');
 
--Результат: 'Taxi'
