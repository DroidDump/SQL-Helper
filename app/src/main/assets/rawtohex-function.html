RAWTOHEX ФУНКЦИЯ

В этом учебном пособии вы узнаете, как использовать Oracle/PLSQL функцию RAWTOHEX с синтаксисом и примерами.

Описание

Функция Oracle/PLSQL RAWTOHEX преобразует значение типа RAW в шестнадцатеричное представление символа.

Синтаксис

Синтаксис функции Oracle/PLSQL RAWTOHEX:

RAWTOHEX( raw )

Параметры или аргументы

raw исходное значение RAW для конвертации в шестнадцатеричное значение.

Функция RAWTOHEX возвращает string значение.

Применение

Функцию RAWTOHEX можно использовать в следующих версиях Oracle/PLSQL:

  • Oracle 12c, Oracle 11g, Oracle 10g, Oracle 9i, Oracle 8i

Пример

Рассмотрим пример функции RAWTOHEX и изучим, как использовать функцию RAWTOHEX в Oracle/PLSQL.

Oracle PL/SQL
1
2
3
4
SELECT RAWTOHEX(HEXTORAW('8a'))
  FROM DUAL;
 
--Результат:   8A
