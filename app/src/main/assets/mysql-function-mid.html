MySQL функция MID

В этом учебном пособии вы узнаете, как использовать MySQL функцию MID с синтаксисом и примерами.

Описание

MySQL функция MID извлекает подстроку из строки.

Синтаксис

Синтаксис MySQL функции MID:

MID( string, start_position, length )

Параметры или аргументы

string - исходная строка, из которой извлекается подстрока.

start_position - начальная позиция для извлечения. Первая позиция в строке всегда равна 1.

length - количество символов для извлечения.

Примечание

  • Первая позиция в строке - 1.
  • Если start_position - положительное число, то функция MID начинается с начала строки.
  • Если start_position - отрицательное число, то функция MID начинается с конца строки и отсчитывает назад.
  • Функция MID и функция SUBSTR являются синонимами функции SUBSTRING.

Применение

Функция MID может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23
