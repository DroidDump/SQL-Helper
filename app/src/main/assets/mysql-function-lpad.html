MySQL функция LPAD

В этом учебном пособии вы узнаете, как использовать MySQL функцию LPAD с синтаксисом и примерами.

Описание

MySQL функция LPAD возвращает строку, дополненную слева с указанной строкой до определенной длины.

Синтаксис

Синтаксис MySQL функции LPAD:

LPAD( string, length, pad_string )

Параметры или аргументы

string - строка для дополнения символов (слева).

length – длина результирующей строки после дополнения слева.

pad_string - строка для дополнения к string слева.

Примечание

  • Если длина строки больше length, то функция LPAD удалит символы из string, и сократит string до length символов.
  • См. также функцию RPAD.

Применение

Функция LPAD может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23
