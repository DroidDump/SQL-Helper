Числовые / математические функции MySQL

Функция & Описание
ABS
MySQL функция ABS возвращает абсолютное значение числа.
ACOS
MySQL функция ACOS возвращает арккосинус числа.
ASIN
MySQL функция ASIN возвращает арксинус числа.
ATAN
MySQL функция ATAN возвращает арктангенс числа.
ATAN2
MySQL функция ATAN2 возвращает арктангенс n и m.
AVG
MySQL функция AVG возвращает среднее значение выражения.
CEIL
MySQL функция CEIL возвращает наименьшее целочисленное значение, которое больше или равно числу.
CEILING
MySQL функция CEILING возвращает наименьшее целочисленное значение, которое больше или равно числу.
COS
MySQL функция COS возвращает косинус числа.
COT
MySQL функция COT возвращает котангенс числа.
COUNT
MySQL функция COUNT возвращает количество выражений.
DEGREES
MySQL функция DEGREES преобразует радианное значение в градусы.
DIV
MySQL функция DIV используется для целочисленного деления, где x делится на y и возвращается целое значение.
EXP
MySQL функция EXP возвращает e, возведенный в степень числа e number .
FLOOR
MySQL функция FLOOR возвращает наибольшее целочисленное значение, которое равно или меньше числа.
GREATEST
MySQL функция GREATEST возвращает наибольшее значение в списке выражений.
LEAST
MySQL функция LEAST возвращает наименьшее значение в списке выражений.
LN
MySQL функция LN возвращает натуральный логарифм числа.
LOG
MySQL функция LOG возвращает либо натуральный логарифм числа, если вызывается с одним параметром, либо логарифм числа по основанию, если вызывается с двумя параметрами.
LOG10
MySQL функция LOG10 возвращает десятичный логарифм числа.
LOG2
MySQL функция LOG2 возвращает логарифм числа по основанию 2.
MAX
MySQL функция MAX возвращает максимальное значение выражения.
MIN
MySQL функция MIN возвращает минимальное значение выражения.
MOD
MySQL функция MOD возвращает остаток от деления n на m.
PI
MySQL функция PI возвращает значение π (Pi) отображается с 6 знаками после запятой.
POW
MySQL функция POW возвращает m, возведенную в степень n.
POWER
MySQL функция POWER возвращает m, возведенную в степень n.
RADIANS
MySQL функция RADIANS преобразовывает значение из градусов в радианы.
RAND
MySQL функция RAND генерирует случайное число или случайное число в диапазоне.
ROUND
MySQL функция ROUND возвращает число, округленное до определенного количества десятичных знаков.
SIGN
MySQL функция SIGN возвращает значение, указывающее знак числа.
SIN
MySQL функция SIN возвращает синус числа.
SQRT
MySQL функция SQRT возвращает квадратный корень числа.
SUM
MySQL функция SUM возвращает суммарное значение выражения.
TAN
MySQL функция TAN возвращает тангенс числа.
TRUNCATE
MySQL функция TRUNCATE возвращает число, усеченное до определенного количества десятичных знаков.
