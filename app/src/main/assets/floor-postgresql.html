PostgreSQL функция floor

В этом учебном пособии вы узнаете, как использовать PostgreSQL функцию floor с синтаксисом и примерами.

Описание

Функция floor PostgreSQL возвращает наибольшее целочисленное значение, которое меньше или равно number.

Синтаксис

Синтаксис функции floor в PostgreSQL:

floor( number )

Параметры или аргументы

number
Значение, используемое для поиска наибольшего целочисленного значения.

Примечание

  • Также смотрите ceil, ceiling, div, round и trunc.

Применение

Функция floor может быть использована в следующих версиях PostgreSQL:

  • PostgreSQL 11, PostgreSQL 10, PostgreSQL 9.6, PostgreSQL 9.5, PostgreSQL 9.4, PostgreSQL 9.3, PostgreSQL 9.2, PostgreSQL 9.1, PostgreSQL 9.0, PostgreSQL 8.4
