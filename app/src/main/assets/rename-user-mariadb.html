MariaDB оператор RENAME USER

В этом учебном пособии вы узнаете, как использовать оператор RENAME USER с синтаксисом и примерами.

Описание

Оператор RENAME USER используется для переименования пользователя в базе данных MariaDB.

Синтаксис

Синтаксис оператора RENAME USER в MariaDB

RENAME USER
user_name TO new_name;

Параметры или аргументы

user_name
Имя пользователя для переименования в базе данных MariaDB.
new_name
Новое имя, назначаемое пользователю.

Пример

Давайте посмотрим, как переименовать пользователя в MariaDB с помощью оператора RENAME USER.
Например:

PgSQL
1
2
RENAME USER
  'admin'@'localhost' TO 'trizor'@'localhost';

В этом примере оператор RENAME USER переименует пользователя с именем 'admin' в 'trizor' в базе данных MariaDB.

Переименовать более одного пользователя

Вы можете использовать оператор RENAME USER, чтобы переименовать нескольких пользователей, разделяя значения имени пользователя через запятую.
Например:

PgSQL
1
2
3
RENAME USER
  'admin'@'localhost' TO 'trizor'@'localhost',
  'shavka'@'localhost' TO 'adm'@'localhost';

Этот пример переименует двух пользователей в MariaDB. Первый пользователь с именем 'admin' будет переименован в 'trizor', а второй пользователь с именем 'shavka' будет переименован в 'adm'.
