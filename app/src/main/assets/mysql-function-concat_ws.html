MySQL функция CONCAT_WS

В этом учебном пособии вы узнаете, как использовать MySQL функцию
CONCAT_WS с синтаксисом и примерами.

Описание

MySQL функция CONCAT_WS позволяет объединять два или более выражения вместе и добавляет разделитель между каждым конкатенированным выражением.

Синтаксис

Синтаксис MySQL функции CONCAT_WS:

CONCAT_WS( separator, expression1, expression2, ... expression_n )

Параметры или аргументы

separator - разделитель, который добавляется между каждым конкатенированным выражением.

expression1, expression2, ... expression_n - выражения, которые объединяются

Примечание

  • Функция CONCAT_WS пропускает выражения (то есть: expression1, expression2, ... expression_n), которые содержат значение NULL. Если разделитель является NULL, функция CONCAT_WS возвращает значение NULL.
  • См. Также функцию CONCAT.

Применение

Функция CONCAT_WS может использоваться в следующих версиях MySQL:

  • MySQL 5.7, MySQL 5.6, MySQL 5.5, MySQL 5.1, MySQL 5.0, MySQL 4.1, MySQL 4.0, MySQL 3.23

mysql> SELECT CONCAT_WS('_', 'MySQL', 'for', 'you');
 
#Результат:  MySQL_for_you                        
 
mysql> SELECT CONCAT_WS(',', 1, 2, 3, 4);
 
#Результат:  1,2,3,4                    
 
mysql> SELECT CONCAT_WS(', ', 1, 2, 3, 4);
 
#Результат:  1, 2, 3, 4                  
 
mysql> SELECT CONCAT_WS('ABC', 'x', 'y', 'z');
#Результат:  xABCyABCz                      
 
mysql> SELECT CONCAT_WS('ABC', 'x', 'y', NULL, 'z');
 
#Результат:  xABCyABCz                            
 
mysql> SELECT CONCAT_WS(NULL, 'x', 'y', 'z');
 
#Результат:  NULL
