trim функция SQLite

В этом учебном пособии вы узнаете, как использовать SQLite функцию trim с синтаксисом и примерами.

Описание

SQLite функция trim удаляет все указанные символы с обеих сторон строки.

Синтаксис

Синтаксис функции trim в SQLite:

trim( string, [ trim_string ] )

Параметры или аргументы

string
Строка для обрезки символов.

trim_string
Строка, которая будет удалена с обеих сторон string. Если этот параметр не указан, функция trim удалит как начальные, так и завершающие пробелы из string.

Примечание

Смотрите также функции ltrim и rtrim.

Применение

Функция trim может использоваться в следующих версиях SQLite:

  • SQLite 3.8.6, SQLite 3.8.x, SQLite 3.7.x, SQLite 3.6.x
