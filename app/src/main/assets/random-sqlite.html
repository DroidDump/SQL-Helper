random функция SQLite

В этом учебном пособии вы узнаете, как использовать SQLite функцию random с синтаксисом и примерами.

Описание

SQLite функция random используется для возврата случайного числа.

Синтаксис

Синтаксис функции random в SQLite:

random()

Параметры или аргументы

Для функции random нет параметров или аргументов.

Примечание

  • Функция random вернет значение между -9223372036854775808 и +9223372036854775807.

Применение

Функция random может использоваться в следующих версиях SQLite:

  • SQLite 3.8.6, SQLite 3.8.x, SQLite 3.7.x, SQLite 3.6.x
