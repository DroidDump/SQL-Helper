ASCII ФУНКЦИЯ

В этом учебном пособии вы узнаете, как использовать функцию ASCII Oracle/PLSQL с синтаксисом и примерами.

Описание

Функция Oracle/PLSQL ASCII возвращает числовое представление крайнего левого символа строки.

Синтаксис

Синтаксис функции Oracle/PLSQL ASCII:

ASCII( single_character )

Параметры или аргументы

single_character символ для получения числового кода. Если передается более чем один символ, функция ASCII вернет значение для первого символа и будет игнорировать все последующие символы.

  • Функция ASCII возвращает numeric значение.

Применение

Функцию ASCII можно использовать в следующих версиях Oracle/PLSQL:

  • Oracle 12c, Oracle 11g, Oracle 10g, Oracle 9i, Oracle 8i
