PostgreSQL функция rtrim

В этом учебном пособии вы узнаете, как использовать в PostgreSQL функцию rtrim с синтаксисом и примерами.

Описание

Функция PostgreSQL rtrim удаляет все указанные символы в правой части строки.

Синтаксис

Синтаксис функции rtrim в PostgreSQL:

rtrim( string, trim_character )

Параметры или аргументы

string
Строка для обрезки с правой стороны.
trim_character
Набор символов, который будет удален из правой части строки. Если этот параметр пропущен, функция rtrim удалит пробелы в правой части string.

Применение

Функция rtrim может использоваться в следующих версиях PostgreSQL:

  • PostgreSQL 11, PostgreSQL 10, PostgreSQL 9.6, PostgreSQL 9.5, PostgreSQL 9.4, PostgreSQL 9.3, PostgreSQL 9.2, PostgreSQL 9.1, PostgreSQL 9.0, PostgreSQL 8.4
